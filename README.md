# Conversor de Monedas, Java challenge



Esta aplicación Java convierte diferentes tipos de monedas usando una API de tasas de cambio.



&nbsp;Funciones:

\- Conversión entre monedas.

\- Consulta en tiempo real a una API de tipos de cambio.

\- Interfaz por consola amigable.



&nbsp;Tecnologías:

\- Java 17+

\- Librería Gson (para manejar JSON)

\- API de ExchangeRate
